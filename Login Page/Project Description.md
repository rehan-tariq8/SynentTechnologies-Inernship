# Modern Responsive Login Interface

This project is a modern and fully responsive login page developed using HTML, CSS, and Bootstrap 5. The interface is 
designed with a clean and professional layout that focuses on user experience, proper alignment, smooth spacing, and 
responsive behavior across different devices.

The login interface features a split-screen design with a visually attractive left section containing branding and welcoming
text, while the right section contains the authentication form. The project demonstrates modern frontend development 
practices, responsive web design techniques, and structured UI styling.

---

# 🚀 Features

* Modern split-screen login interface
* Fully responsive design for desktop, tablet, and mobile devices
* Clean and professional UI/UX
* Email and password input fields
* “Forgot Password” functionality text
* Styled login button with hover effects
* Proper spacing, alignment, and typography
* Bootstrap 5 integration
* Mobile-friendly responsive layout using media queries
* Reusable and organized CSS styling

# 📱 Responsive Behavior

The project is optimized for:

* Large desktop screens
* Tablets
* Mobile devices

Media queries are used to automatically adjust:

* Layout structure
* Font sizes
* Form container width
* Section positioning
